## Project

Plant Manager is an application under development, initially conceived by <a href="https://www.rocketseat.com.br/"> **Rocketseat** </a> to facilitate plant management in home environments. Using <a href="https://reactnative.dev/"> **React Native** </a> technology, the primary goal is to create a platform that allows users to catalog their plants while also providing functionality for scheduling reminders to water them.

The next step is to expand the application's functionality in future phases, including replication in <a href="https://www.apple.com/br/swift/"> **Swift (IOS)** </a>, so that I can demonstrate proficiency in the respective languages and highlighting versatility in different development environments.

O meta é expandir a funcionalidade do aplicativo em fases futuras, incluindo a replicação em <a href="https://www.apple.com/br/swift/"> **Swift(IOS)** </a>, demonstrando assim proficiência nas respectivas linguagens e destacar a versatilidade em ambientes de desenvolvimento distintos.

#### v1.0:
- [ ] Listagem de plantas;
- [ ] CRUD das plantas;
- [ ] Recebimento de alertas: Alertas para regar;
- [ ] Autenticação com o nome: Utilizar o nome para acessar as plantas por perfil.
- [ ] Notificações de Eventos Especiais (Aniversário da Planta): Celebração de datas especiais, como aniversários de plantas, com notificações personalizadas e sugestões de presentes ou cuidados adicionais.

#### v2.0:
- [ ] Lembrentes personalizados: Adicionar a possibilidade de adiconar lembretes para;: Regar, Adubar, Podar, etc; 
- [ ] Histórico de Atividades: Rastreio e exibição do histórico de atividades dos usuários, como a última vez que realizaram uma rega, adubação ou atualizaram informações sobre uma planta.
- [ ] Filtro e Pesquisa Avançada: Filtro e pesquisa para que os usuários possam encontrar suas plantas com base em critérios específicos, como nome, tipo ou data de última rega.
- [ ] Compartilhamento em Redes Sociais: Compartilhamento de fotos e informações sobre suas plantas em suas redes sociais, promovendo a interação com a comunidade de amantes de plantas.
- [ ] Autenticação completa: Utilizando email pessoal e possibilidade de acessar com a conta Google.

#### v3.0:
- [ ] Sistema de Recompensas e Conquistas: Sistema de recompensas baseado em conquistas, incentivando os usuários a cuidar ainda melhor de suas plantas.
- [ ] Integração com o Calendário: Criar lembretes para comprar suprimentos.
- [ ] Registro de Eventos Especiais:  Possibilidade dos usuários registrar eventos importantes, como replantio, poda ou florescimento, para um acompanhamento mais completo.
- [ ] Relatórios e Estatísticas de Crescimento: Apresentação de gráficos e relatórios que demonstrem o progresso de cada planta ao longo do tempo, incluindo informações sobre altura, folhagem e saúde.
- [ ] Integração com Previsão do Tempo: Informações sobre as condições meteorológicas locais para ajudar os usuários a adaptar os cuidados de suas plantas de acordo.

#### v4.0:
- [ ] Biblioteca de Artigos sobre Jardinagem: Integração uma seção com artigos e guias informativos sobre cuidados com plantas, dicas de cultivo e solução de problemas comuns.
	- [ ] Dicas de Companhia para Animais de Estimação: Informações sobre plantas seguras para ter em ambientes com animais de estimação, e alertas sobre plantas tóxicas.
	- [ ] Conselhos de Cuidados, Adubação e Fertilização: Orientações específicas sobre quando e como fertilizar cada tipo de planta, com lembretes configuráveis.
	- [ ] Catálogo de Plantas Externas: Integração com um catálogo abrangente de plantas, permitindo aos usuários adicionar novas espécies ao seu perfil, juntamente com informações 	relevantes sobre cuidados e características.
- [ ] Integração com Assistente de Voz: Implementação da capacidade dos usuários interagirem com o aplicativo por meio de comandos de voz, facilitando a entrada de dados e a realização de ações.
- [ ] Compatibilidade com Dispositivos IoT: Integração do aplicativo com dispositivos de IoT para monitorar e controlar variáveis como temperatura, umidade e luminosidade do ambiente das plantas.
- [ ] Acesso Offline e Sincronização em Nuvem: Aplicativo acessível mesmo sem conexão à internet, e permitir a opção de sincronização em nuvem para manter os dados atualizados em todos os dispositivos.

## Installation

#### Clone the Repository
```bash
git clone https://github.com/WesleyReboucas/PlantManager.git
cd PlantManager/
```
##### React Native - IOS
```bash
npm i && npx expo run:ios
```
##### React Native - Android
```bash
npm i && npx expo run:android
```
